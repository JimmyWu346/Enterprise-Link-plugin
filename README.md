# 企业关系递归导出插件

## 功能

- 在目标站点页面右下角提供“企业链路导出”按钮。
- 从地址栏读取 `orgid` 参数作为一级公司入参。
- 按两条链路递归抓取：
  - 股东分支：只取唯一最大股东；若并列最大则不继续。
  - 对外投资分支：递归抓取企业节点。
- 仅一级公司抓取“主要人员”，其它层级不抓取主要人员。
- 自动跳过重复个人与重复公司名称。
- 一键导出 Excel（`.xls`，多 sheet）：
  - `一级公司人员信息`
  - `企业关系链路`

## 安装

1. 打开 Chrome 扩展管理页：`chrome://extensions/`
2. 开启“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录

## 说明

- 仅在 `https://zsk-cmis.cicpa.org.cn/*` 生效。
- 接口需用户当前登录态可用（脚本使用 `credentials: include`）。
